# LangChain with Generative AI

This repository contains hands-on Python code examples demonstrating how to use **LangChain with Generative AI**.  
The code is written for learning and experimentation, starting from basic concepts and progressing to complete end-to-end applications.

The repository focuses on **clear implementation**, **practical usage**, and **concept-wise separation** to make understanding LangChain easier.

---

## Contents Covered

The repository includes examples for the following LangChain concepts:

- Interacting with LLMs (API-based and Local models)
- Chat Models
- Prompt Templates
- Structured Outputs
- Output Parsers
- Runnables and Chains
- Document Loaders
- Text Splitters
- Embeddings
- Vector Stores
- Retrievers
- Retrieval Augmented Generation (RAG)
- Tool creation and Tool Calling
- Agent-based workflows
- End-to-End GenAI Projects
